# My Dotfiles Repo

This repo includes dotfiles for

- `zsh`
- `vim`
- `brew`

# How to install
```
zsh -c "$(curl -fsSL https://raw.githubusercontent.com/schroneko/dotfiles/main/install.sh)"
```

# Structure
```
❯ tree dotfiles -a -I '.git'    

dotfiles
├── .Brewfile
├── .config
│   └── starship.toml
├── .hushlogin
├── .vimrc
├── .zshrc
├── README.md
└── install.sh
```
