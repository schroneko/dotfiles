# My Dotfiles Repo

This repo includes dotfiles for

- `zsh`
- `vim`
- `brew`

# How to install
```
curl -o - https://raw.githubusercontent.com/schroneko/dotfiles/main/install.sh | zsh
```

# Structure
```
❯ tree dotfiles -a -I '.git'    
dotfiles
├── .Brewfile
├── .vimrc
├── .zshrc
├── README.md
└── install.sh
```
